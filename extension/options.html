勾选要参与同步的网页 AI；每个站点的选择器都可编辑——网站改版导致"不发送/找不到输入框"时，按 F12 找到对应元素改这里即可，无需改代码。

关于 PromptSync

PromptSync 无后端、无统计或埋点，不收集聊天内容。同步任务仅在浏览器本地处理，并直接发送到你勾选的 AI 网站。

作者主页使用 GitHub Profile，项目建议与问题统一在 GitHub Issues 留言，方便公开跟踪和后续维护。
