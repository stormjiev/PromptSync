# PromptSync — 多 AI 同步输入（独立 Chrome 扩展）

把 `dual-ai-sync.user.js`（Tampermonkey 用户脚本）升级为**独立浏览器扩展**：
零依赖 Tampermonkey、一键安装、可视化配置站点、支持多家网页 AI 同步发送。

> 这是主仓库脚本的"发散版本"。原 `../dual-ai-sync.user.js` 保持不变。

## 它能做什么

- 在任意已启用的 AI 页面右下角弹出浮动小框，输入/粘贴/拖拽（多图多文件），
  回车后**同步发往你勾选的所有网页 AI**并自动发送。
- 内置支持：**ChatGPT、Gemini、DeepSeek、Qwen（通义千问）**。
- 保留用户脚本里来之不易的稳定性逻辑：
  - 附带文件时**等上传真正完成再发送**（网络层 fetch/XHR 钩子 + DOM 指示器双保险）。
  - **单任务只点一次发送键**，绝不自动重试 → 杜绝重复发送 / 截断回答。
  - 跨标签页 / 跨实例防重锁。
- **可视化配置**：每个站点的选择器都能在设置页编辑；网站改版只需改选择器，不用动代码。
- **新增任意网页 AI**：填域名 + 选择器，授权后动态生效。

## 安装（开发者模式加载）

1. 打开 `chrome://extensions`
2. 右上角开启「开发者模式」
3. 点「加载已解压的扩展程序」，选择本 `extension/` 目录
4. 打开任意受支持的 AI 网站并保持登录即可看到浮动面板

> 后续要上架 Chrome 应用商店：把本目录打包为 zip 上传即可（已是标准 MV3 结构）。

## 使用

- 面板顶部勾选要同步的目标站点（默认全选已启用站点）。
- 输入框：**回车 = 同步发送**，**Shift+Enter = 换行**。
- 三个按钮：同步填入（只填不发）/ 同步发送（当前对话）/ 新窗口发送（各开新对话再发）。
- 点击右上角 ⚙ 打开设置页；点工具栏图标弹窗可快速开关站点。

## 配置其它网页 AI

设置页（⚙）→「➕ 新增站点」：
- **域名关键字**：如 `chat.deepseek.com`
- **输入框 / 发送按钮 / 用户消息气泡 / 上传指示器 / 停止按钮** 等选择器（F12 查元素填）
- 保存时浏览器会弹窗请求该域名的访问权限，授权后即可生效（无需重新打包）。

内置站点选择器留空＝沿用默认；改版失效时填入新的选择器即可即时修复。

## 已知限制

- DeepSeek / Qwen 的默认选择器为尽力而为，若失效请在设置页按 F12 调整。
- 跨标签页防重锁基于异步存储，极端并发下仍以页内 `sentTokens` 为主防线。
- 大文件经 base64 在 `chrome.storage` 间传递，建议单文件控制在数 MB 内。
- 大规模自动化可能触及各家服务条款，仅供个人少量使用。

## 文件结构

```
extension/
  manifest.json          MV3 清单
  src/
    inject-main.js       MAIN 世界：网络层上传跟踪
    adapters.js          站点预设（数据驱动，options/popup/content 共用）
    content.js           核心：同步 + 发送防重 + 浮动面板
    background.js        service worker：打开设置 + 自定义站点动态注册
  options.html / options.js   可视化配置页
  popup.html / popup.js       工具栏快速开关
  icons/                 图标
```
